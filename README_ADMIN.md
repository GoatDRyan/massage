# Renaissens — admin gratuit

L'interface admin personnalisée est disponible sur :

```txt
/admin
```

Elle permet de modifier le contenu du site sans Sanity ni CMS externe.

Avant utilisation en ligne, configure les variables Vercel indiquées dans :

```txt
docs/FREE_ADMIN_SETUP.md
```

En local, l'interface permet déjà de prévisualiser et télécharger le fichier `content.json`, mais la sauvegarde GitHub fonctionne uniquement quand les variables Vercel sont configurées.
